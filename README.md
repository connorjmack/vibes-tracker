## Project Overview
This AI agent is designed to monitor and analyze narrative trends across custom-defined clusters of YouTube channels (e.g., "Tech Reviewers," "Manosphere").

It uses:
* YouTube Data API for ingestion.
* LLMs (Gemini/GPT) for intelligent topic extraction and summarization.
* RAG (Retrieval-Augmented Generation) for deep, searchable analysis of video transcripts.
